Website Stock Availability Filter

This module will help the customers to have a quick filter for the available products in website. It will give an option to filter out the stock available products in the website.

Get Product Availability Filter

As shown, you will be able to see the stock filter in the attribute and filter section and also "Available" label will be displayed on stock products.

When clicking on the filter it will display products that are available in stock.

This filter also works when you are in a specific category it will display products that are available in stock for a specific category.
